# Dataset catalog

The canonical machine-readable catalog is `configs/datasets.yaml`. The MVP is anchored to the official ALMA Figshare workbook (DOI `10.6084/m9.figshare.17212856.v1`). Its downloaded payload contains figure/source-data worksheets, so CoRe-TBI inventories it without misrepresenting it as a complete pose or animal-level session table. Downloaded contents must be inventoried before analysis; source records and access terms are authoritative.
