# Prospective repetitive-mild-TBI rat protocol

Suggested recordings: baseline, 1, 3, 7, 14, and 30 days after injury, across open field, avoidance, elevated-plus maze, home cage, and a feasible gait/ladder assay. Store stable `animal_id`, session, task, injury, sex, age, camera view, frame rate, treadmill speed, and synchronization clock fields.

Suggested keypoints: nose, bilateral ears, neck, mid-spine, bilateral shoulders/forepaws/hips/hind paws, tail base/middle/tip. Future neural interfaces should accept timestamped LFP features (prelimbic/infralimbic PFC, BLA, hippocampus, theta coherence, gamma power, cross-frequency coupling) through a separate table keyed by animal/session/time. Neural data are explicitly out of scope for the MVP.
