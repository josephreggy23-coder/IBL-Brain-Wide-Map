# Scientific rationale

CoRe-TBI tests whether post-injury task performance and movement normalization separate. It compares each observation with that animal's pre-injury reference, rather than treating a group-level injury classifier as recovery. The Phase 1 model calculates robust multivariate movement deviation and performance deviation with tolerances estimated from baseline repeatability and sham observations.

This framework cannot prove a biological compensatory mechanism from observational pose data. It supplies an explicit, falsifiable operational proxy and uncertainty-aware reports.
